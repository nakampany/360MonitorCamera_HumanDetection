# 機能概要

</br>
thetaV,openCV
を使用し、360度監視します



# 機能一覧


openCVで顔を検知をする</br>
顔が検出されたら、thetaで撮影をする</br>
撮影されたimgはWEB上にアップロードされ、閲覧できる</br>
